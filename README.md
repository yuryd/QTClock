# QTClock

Clock running under QT5: Windows, Linux

Dependencies:
(PyQt5, geopy, tzlocal, geocoder)

Preparation:
cd cloned directory

python -m venv venv
(Windows) call venv\Scripts\activate.bat
(Linux) source venv/Scripts/activate

Dependencies:
pip install PyQt5
pip install geopy
pip install tzlocal
pip install geocoder
